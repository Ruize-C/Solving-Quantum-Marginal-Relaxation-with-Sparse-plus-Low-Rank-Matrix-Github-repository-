# Solving Quantum Marginal Relaxation with Sparse plus Low-Rank Matrix

This repository contains the code used for two 2D models discussed in the paper:

- `TFI 2D/`: MATLAB scripts for the 2D transverse-field Ising model workflow, plus small-system DMRG ground-state energy estimation code.
- `Hubbard 2D/`: MATLAB scripts for the 2D Hubbard model workflow, plus small-system DMRG ground-state energy estimation code.

## Directory layout

Each model directory follows the same structure:

- `data/`: fixed operators and generated intermediate data files.
- `script/`: main MATLAB scripts and Julia notebooks for small-scale ground-state energy estimation.
- `benchmark_result/`: benchmark outputs saved by `alg4.m`.

## Recommended run order

Inside either `TFI 2D/` or `Hubbard 2D/`, run:

1. `script/prepare_data.m`
2. `script/init_S0.m`
3. `script/alg4.m`

## Quick Start

For either model directory:

1. Open MATLAB in `TFI 2D/` or `Hubbard 2D/`.
2. Run `script/prepare_data.m`.
3. Run `script/init_S0.m`.
4. Run `script/alg4.m`.
5. Check `benchmark_result/<N>_benchmark.mat` for the saved benchmark output.

Expected outputs:

- `data/<N>_data.mat`: fixed matrices, reference energies, and other data generated by `prepare_data.m`.
- `data/<N>_S0.mat`: initialization data for the sparse plus low-rank factorization generated by `init_S0.m`.
- `benchmark_result/<N>_benchmark.mat`: convergence curves and benchmark metrics saved by `alg4.m`.
